# BrighterVision Backend

เซิร์ฟเวอร์ Backend สำหรับระบบไม้เท้าและแว่นตาอัจฉริยะ

## วิธีการติดตั้งและรันเซิร์ฟเวอร์

1. **ติดตั้งไลบรารีที่จำเป็น** (เข้าไปในโฟลเดอร์ `backend/` ก่อน)

    ```bash
    pip install -r requirements.txt
    ```

2. **เปิดการทำงานเซิร์ฟเวอร์**
    ```bash
    python main.py
    ```
    _เซิร์ฟเวอร์จะรันที่ `http://localhost:3000` (มี IP บอกใน Console สำหรับให้อุปกรณ์ต่อ WiFi เดียวกันเรียกใช้งาน)_

---

## เส้นทาง API (Routes)

### 1. ตรวจจับวัตถุ (Object Detection)

- **Endpoint:** `POST /button`
- **ส่งข้อมูลมา:** ไฟล์รูปภาพ RAW Binary
- **ตอบกลับ:** ไฟล์เสียง (`.wav`) บอกชื่อวัตถุที่ตรวจพบหรือบอกว่าไม่พบ (ส่งไฟล์มาให้เล่นโดยตรง)
  _(หมายเหตุ: ต้องปลดคอมเมนต์ `app.include_router(object_detection_router)` ใน `main.py` ก่อนเพื่อเปิดใช้งาน)_

### 2. อ่านข้อความจากภาพ (Text Detection / OCR)

- **Endpoint:** `POST /ocr`
- **ส่งข้อมูลมา:** ไฟล์รูปภาพ RAW Binary
- **ตอบกลับ:** ไฟล์เสียง (`.wav`) เสียงอ่านข้อความที่เจอในรูปภาพ (ส่งไฟล์มาให้เล่นโดยตรง)

### 3. ส่งพิกัดตำแหน่ง (GPS)

- **Endpoint:** `GET /gps?lat=...&lng=...`
- **ตอบกลับ:** JSON ยืนยันว่าได้รับพิกัดแล้ว

### 4. เช็คสถานะการทำงานของเซิร์ฟเวอร์ (Health Check)

- **Endpoint:** `GET /health`
- **ตอบกลับ:** JSON แจ้งว่าเซิร์ฟเวอร์ทำงานปกติ
