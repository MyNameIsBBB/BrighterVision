# BrighterVision 🕶️🦯

**Assistive Technology Ecosystem for Visually Impaired Individuals**

BrighterVision คือโปรเจกต์พัฒนาระบบนิเวศเทคโนโลยีสิ่งอำนวยความสะดวกสำหรับผู้พิการทางสายตา ประกอบด้วย **แว่นตาอัจฉริยะ (Smart Glasses)** และ **ไม้เท้าอัจฉริยะ (Smart Cane)** ที่ทำงานร่วมกับระบบประมวลผล Cloud-based AI เพื่อช่วยในการระบุวัตถุและนำทางได้อย่างปลอดภัย

## 💡 ฟีเจอร์หลัก (Key Features)

- **Object Detection:** ตรวจจับวัตถุ กีดขวาง และป้ายสัญลักษณ์ในระยะเรียลไทม์โดยใช้ YOLOv26n
- **Audio Feedback:** แจ้งเตือนสถานะและตำแหน่งวัตถุผ่านเสียง (Text-to-Speech)
- **Smart Navigation:** ระบบนำทางอัจฉริยะที่วิเคราะห์เส้นทางร่วมกับเซ็นเซอร์ Ultrasonic บนไม้เท้า
- **Emergency Alert:** ระบบส่งพิกัด GPS ไปยังผู้ดูแลเมื่อเกิดอุบัติเหตุหรือมีการกดปุ่มฉุกเฉิน

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Hardware:** ESP32, ESP32-Camera Module, Ultrasonic Sensors, MPU6050
- **AI/ML:** Python, PyTorch, YOLOv26n
- **Backend:** Next.js (Dashboard สำหรับผู้ดูแล), FastAPI (API & AI Processor)
